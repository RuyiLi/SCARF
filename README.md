# SCARF

A **S**imple **C**ommand **A**nd **R**esponse **F**ramework for the node.js Discord API wrapper, [discord.js](https://discord.js.org/#/). 

## Installation

With npm:
```
npm i scarf-djs
```

With yarn:
```
yarn add scarf-djs
```

## Usage
See the test folder for an example bot created using SCARF.
